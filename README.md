# DailyDoze
An Android app that lets you co-relate your sleep, weight and food habits and helps you improve it.

## Technologies Used :
a. Java<br>
b. XML<br>
c. Groovy<br>
d. Firebase(Database)

**Developed on Android Studio**
<br>
<br>

## Features :
a. Daily Doze<br>
b. 21 Tweaks<br>
c. Calender & Info<br>
d. Fast Watch<br>
e. Sleep and Weight Tracker<br>
f. Meditation<br>
g. Notification<br>
h. Record History<br>
i. Sign Up and Log In
<br>
<br>
